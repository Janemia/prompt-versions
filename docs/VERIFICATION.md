## 四、端到端测试流程

### 场景 1：创建并管理 Prompt

```
1. 访问 Web UI 首页
2. 点击"新建 Prompt"
3. 输入名称：customer-support
4. 输入 Prompt 内容
5. 点击"创建"
6. ✓ 看到新 Prompt 出现在列表中
7. 点击"查看"进入详情页
8. ✓ 看到 v1 版本
9. 修改内容并保存（通过 CLI）
10. 刷新页面
11. ✓ 看到 v2 版本
12. 点击"对比"
13. 选择 v1 和 v2
14. ✓ 看到差异对比
```

### 场景 2：测试 Prompt

```
1. 访问测试中心
2. 点击"添加测试"
3. 输入测试名称、输入、期望输出
4. ✓ 测试用例出现在列表中
5. 点击"运行测试"
6. ✓ 看到测试结果
```
