## 技术实现

### 方案选择

**为什么使用内联样式？**
- ✅ 绕过 CSS 编译问题
- ✅ 完全控制样式
- ✅ 无构建配置问题
- ✅ 类型安全 (TypeScript)

**样式对象结构**:
```typescript
const styles = {
  page: { CSSProperties },
  header: { CSSProperties },
  // ...
};
```

### 响应式设计

```typescript
cardGrid: {
  display: 'grid',
  gridTemplateColumns: 'repeat(auto-fill, minmax(300px, 1fr))',
  gap: '1.5rem',
}
```

### 动画效果

```typescript
card: {
  transition: 'transform 0.3s',
  // hover 时 translateY(-4px)
}
```
